# 后端接口设计

## 1. 健康检查

GET `/api/health`

返回：

```json
{"status": "ok"}
```

## 2. 规则重载

POST `/api/rules/reload`

作用：
- 清空后端缓存的规则；
- 重新从 MaxKB 按 RULE_ID 加载所有规则和 Prompt。

返回：

```json
{
  "success": true,
  "loaded_rule_count": 24
}
```

## 3. 开始训练

POST `/api/training/start`

请求：

```json
{
  "training_type": "初购转化",
  "difficulty": "中等",
  "customer_type": "随机"
}
```

返回：

```json
{
  "session_id": "uuid",
  "visible_brief": {
    "training_title": "",
    "training_type": "",
    "difficulty": "",
    "exam_scope": [],
    "min_rounds": 10,
    "trainee_notice": ""
  },
  "first_customer_message": "",
  "state": {
    "round_count": 0,
    "min_rounds": 10,
    "current_stage": "opening",
    "emotion_label": "平稳",
    "can_finish": false
  }
}
```

## 4. 发送消息

POST `/api/training/chat`

请求：

```json
{
  "session_id": "uuid",
  "message": "新人回复内容"
}
```

返回：

```json
{
  "customer_reply": "",
  "state": {
    "round_count": 1,
    "min_rounds": 10,
    "current_stage": "",
    "emotion_label": "",
    "can_finish": false
  }
}
```

## 5. 结束训练

POST `/api/training/finish`

请求：

```json
{
  "session_id": "uuid"
}
```

返回：

```json
{
  "result": "成交/未成交",
  "score": 0,
  "is_pass": false,
  "dimension_scores": {},
  "customer_pain_points": [],
  "strengths": [],
  "weaknesses": [],
  "key_turning_points": [],
  "deal_reason": "",
  "lost_reason": "",
  "compliance_risks": [],
  "next_training_focus": [],
  "suggested_better_replies": []
}
```

## 6. 重置训练

POST `/api/training/reset`

请求：

```json
{
  "session_id": "uuid"
}
```

返回：

```json
{
  "success": true
}
```
