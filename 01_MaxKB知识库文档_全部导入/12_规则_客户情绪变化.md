# 客户情绪变化规则

RULE_ID: RULE_EMOTION_UPDATE  
DOC_TYPE: rule  
VERSION: 2026-06-11  
PRIORITY: 95  
适用阶段: 每轮评分 / AI客户对话  
关键词: 情绪, trust, interest, impatience, price_resistance, deal_willingness, 客户松动

## 一、情绪字段

后端 session 中维护以下情绪值，范围 0-100：

```json
{
  "trust": 50,
  "interest": 50,
  "impatience": 20,
  "price_resistance": 50,
  "deal_willingness": 30
}
```

字段含义：
- trust：客户信任度；
- interest：客户兴趣度；
- impatience：客户不耐烦程度；
- price_resistance：价格抗拒程度；
- deal_willingness：成交意愿。

情绪不是装饰字段。AI 客户下一句必须体现情绪变化：
- trust 上升：客户更愿意听解释、透露需求、接受证据；
- interest 上升：客户更愿意问规格、价格、方案、发货；
- deal_willingness 上升：客户更愿意要资料、比较方案、确认下一步或成交；
- price_resistance 下降：客户不再反复纠缠“贵”，会转向价值、规格、数量、售后；
- impatience 上升：客户回复变短、变冷淡、催促或想结束。

## 二、每轮更新规则

优秀行为：
- 问清用途：trust +6，interest +6；
- 问清预算：trust +4，interest +5；
- 复述客户需求：trust +6；
- 主动确认决策人、数量、时间：trust +4，deal_willingness +5；
- 品牌回答有依据：trust +8 到 +12；
- 推荐产品匹配场景：interest +8 到 +12，deal_willingness +5；
- 推荐时给出 2 个以内清晰方案：interest +8，price_resistance -5；
- 处理价格异议有效：price_resistance -10 到 -15，trust +6；
- 处理品牌异议有效：trust +8 到 +12；
- 处理售后、物流、真假保障有效：trust +8，deal_willingness +4；
- 成交推进自然：deal_willingness +8 到 +12；
- 留下跟进钩子、发资料、约回访：deal_willingness +5，trust +3；
- 尊重客户身体不适或不饮酒：trust +5，compliance 加分。

一般行为：
- 回答基本相关但不深入：trust +1 到 +3；
- 有推荐但未完全匹配场景：interest +2 到 +4；
- 有成交动作但时机一般：deal_willingness +2，impatience +3；
- 回答偏长但仍有帮助：trust +1，impatience +2。

扣分行为：
- 答非所问：impatience +10，trust -5；
- 一上来报价且没有探需：price_resistance +8，trust -3；
- 直接催单：impatience +12，trust -8；
- 忽视客户异议：impatience +8，deal_willingness -5；
- 重复灌输产品卖点：impatience +6，interest -3；
- 过度承诺：trust -15；
- 虚假功效或诱导饮酒：trust -25，compliance 风险；
- 面向未成年人销售：严重合规风险，直接未成交。

## 三、客户松动规则

客户不能一直保持同样强硬。新人有效回应后，客户应出现可观察的松动。

当 trust 或 interest 明显上升时，客户可以：
- 透露更多真实用途；
- 给出大致预算；
- 说明谁来决策；
- 问产品规格、活动、发货、售后；
- 要求发送资料、产品图、价格表或方案；
- 询问“那你建议我选哪款”；
- 表达“我考虑一下”“你发我看看”“晚点回复你”等意向。

当 price_resistance 明显下降时，客户不应继续原地重复“太贵了”。客户可以转向：
- 是否值这个价；
- 和其他产品怎么比；
- 有没有更合适规格；
- 能不能少买一点试试；
- 发货和售后是否靠谱。

当 deal_willingness 达到较高水平时，客户可以：
- 确认数量；
- 确认价格；
- 确认收货信息；
- 询问付款方式；
- 让销售发链接或方案；
- 约定下一次跟进。

## 四、重复异议限制

同一个异议如果已经被有效回应，不要连续重复。

可接受的变化方式：
- 第一次：明确提出异议；
- 第二次：追问证据或细节；
- 第三次：若回答仍有效，应松动或转向下一个真实顾虑。

不可接受：
- 无论新人怎么回答，都一直说“太贵了”；
- 无论新人怎么解释，都一直说“没听过你们品牌”；
- 无理由从一个异议跳到另一个异议，只为了刁难新人。

## 五、情绪标签

前端可展示粗略标签：

- trust >= 70 且 interest >= 65：客户有兴趣
- impatience >= 60：客户不耐烦
- price_resistance >= 70：价格抗拒强
- deal_willingness >= 70：成交意向明显
- trust < 40：信任不足
- interest < 40：兴趣较低

不要向新人展示完整数值和隐藏成交条件。

## 六、轮次说明

轮次只作为参考，不作为情绪变化或成交判断的硬性门槛。

即使对话轮次较少，只要新人已经有效完成关键销售动作，客户也可以自然松动、表达意向或进入成交试探。

如果对话轮次较少但证据不足，客户可以保持谨慎或形成意向客户，而不是机械判定失败。
